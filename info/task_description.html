SEND + MORE = MONEY

Given a string consisted of three words "FOO BAR BLAH", each word is a decimal number, and characters are distinct digits. You have to check if FOO + BAR = BLAH has solution or not. For example, "SEND MORE MONEY" has a solution S=9, E=5, N=6, D=7, M=1, O=0, R=8, E=5, M=1, O=0, N=6, E=5, Y=2, ie. SEND + MORE = MONEY is 9567 + 1085 = 10652.

Input: String, a string with three words

Output: Boolean, if there is solution or not

Example: 

    assert checkio("SEND MORE MONEY") == True
    assert checkio("GET LESS JOBS") == True
    assert checkio("ONE TWO THREE") == False

Precondition:

  • The maximum length of string is less than 20.
  • There are less than 10 different letters in one case.
